# Native

A [react-native](https://reactnative.dev/) app built using [expo](https://docs.expo.dev/)

## UI Copy Rules

- Mặc định mọi UI copy tiếng Việt phải dùng đầy đủ dấu câu và dấu tiếng Việt tự nhiên.
- Chỉ dùng chuỗi không dấu khi có ràng buộc kỹ thuật rõ ràng như slug, key nội bộ, tên file, hoặc giới hạn từ hệ thống ngoài.
